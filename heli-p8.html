Project Challenge

Design a PID controller for the elevation axis of a Quanser 3DOF helicopter that meets strict specs: rise time <1s, overshoot <10%, settling time ≈5s, and zero steady-state error. The challenge was translating control theory into parameters that actually work on the physical hardware.

Performance Requirements

Hit all performance specs simultaneously while keeping the system stable. Had to use systematic design methods with root locus analysis and validate everything empirically.

System Identification

Experimental Testing

Step response tests to figure out system characteristics and build transfer function models

Parameter Extraction

Logarithmic decrement method for damping ratio, oscillation analysis for natural frequency

Model Validation

Compared first and second-order transfer functions against experimental data

Design Foundation

Used system parameters as the starting point for PID controller design

System Characteristics

Dynamic Properties

  • Damping ratio: ζ ≈ 0.065 (very underdamped)
  • Natural frequency: ωₙ ≈ 0.9 rad/s
  • System gain: k ≈ 10.18°
  • Time constant: τ ≈ 1.23 s

Transfer Function

Second-order model with oscillatory behavior and low damping. The underdamped nature meant I had to be careful with controller design to avoid excessive overshoot while still meeting rise time requirements.

Controller Design

Used root locus design to systematically place closed-loop poles for the desired response. This approach translates performance requirements into pole placement constraints, then finds PID parameters to achieve those pole locations.

Design Process

Requirement Translation

Overshoot <10% → damping ratio >0.59; settling time <5s → real part <-0.8

Pole Placement

Root locus analysis to find controller gains for desired dominant pole locations

PID Structure

Zeros at s = -1, -2 with integrator pole for both transient and steady-state performance

Fine Tuning

Manual adjustment from root locus starting point to optimize performance trade-offs

Final Parameters

Proportional gain Kₚ ≈ 14.7, Integral gain Kᵢ ≈ 4.9, Derivative gain Kd ≈ 2.45. These values hit all performance specs while maintaining good stability margins.

Results

Performance Achievement

  • Rise time: 0.20 s (target: <1 s) ✓
  • Overshoot: 7% (target: <10%) ✓
  • Settling time: 2.6 s (target: ≈5 s) ✓
  • Steady-state error: ~0% (target: 0%) ✓

Validation

  • Bode plots: Good gain and phase margins
  • Root locus: Closed-loop poles where expected
  • Robustness: Performance held up under parameter variations
  • Hardware test: Worked on the actual helicopter

What I Learned

This project taught me the systematic approach to control system design, from identifying the system through implementing the controller. Root locus provided a structured way to translate performance requirements into controller parameters, while empirical tuning dealt with the practical realities.

Key insights included understanding trade-offs between different performance metrics, recognizing how important accurate system identification is, and getting hands-on experience with classical control design techniques that apply across many engineering applications.

Control Fundamentals

The project reinforced core concepts: proportional action for speed, integral action for accuracy, and derivative action for stability. Understanding how these individual contributions work together was essential for achieving the multi-objective performance requirements.
