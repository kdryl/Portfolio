Project Challenge

Design and program an autonomous robot capable of following a black line across complex tracks including straight segments, curves, intersections, and end-of-line detection—all without human intervention after initial activation.

Technical Requirements

Navigate varying track geometries, handle intersections and decision points, maintain consistent speed and accuracy, implement dead reckoning for position estimation, and demonstrate repeatable autonomous performance.

Technical Implementation

Sensor System

5-sensor infrared array with analog threshold calibration for reliable line detection

Control Algorithm

Weighted proportional steering system replacing basic bang-bang control for smooth navigation

State Management

Finite state machine handling track segments, intersections, and behavioral transitions

Position Estimation

Encoder-based dead reckoning for distance tracking and decision-making support

Control System Evolution

Development progressed from simple bang-bang control to sophisticated weighted proportional steering. The final algorithm converts sensor readings into normalized position estimates, enabling smooth steering adjustments that significantly reduced oscillations compared to binary on/off control methods.

Control Algorithm Comparison

Bang-bang vs. Weighted Proportional Control - showing reduced oscillation and improved path accuracy

Advanced Features

Finite State Machine

Structured approach to track navigation including start box departure, straight line following, curve handling, intersection detection, and end-of-line recognition. Each state optimized for specific track conditions.

Sensor Integration

Analog infrared sensor array with adaptive threshold calibration. Real-time sensor fusion provides robust line position estimation even under varying lighting conditions and track surface variations.

Position Estimation System

Implemented encoder-based dead reckoning to track robot position along the course. This enabled distance-based decision making beyond simple sensor inputs, supporting complex navigation behaviors like intersection counting and end-of-track detection.

Engineering Impact

This project demonstrated practical application of embedded systems design, sensor integration, and autonomous control algorithms. The progression from basic to advanced control methods illustrated the importance of iterative development and systematic testing in robotics applications.

Key technical contributions included developing robust sensor calibration procedures, implementing weighted control algorithms for improved performance, and designing modular state machine architecture that could adapt to different track configurations without reprogramming.

Real-World Applications

The techniques developed—autonomous navigation, sensor fusion, and adaptive control—directly apply to larger robotics systems including warehouse automation, agricultural robots, and autonomous vehicles. The project reinforced the importance of reliable sensor processing and robust control design under varying environmental conditions.
